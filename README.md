### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 图书信息
5. 大厦公告
6. 给、后台管理
7. 个人中心
8. 我的收藏


**管理员功能：**
1. 登录
2. 首页（图书统计、图书信息总数、反馈类型统计）
3. 个人中心（修改密码、个人信息）
4. 用户管理
5. 图书分类管理
6. 图书信息管理
7. 图书借阅管理
8. 图书续借管理
9. 图书归还管理
10. 反馈类型管理
11. 需求反馈管理
12. 归还入库管理
13. 系统管理(系统简介、轮播图管理、大厦公告、关于我们)
